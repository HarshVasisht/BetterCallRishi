Recipe Rating Prediction

A machine learning project developed during my studies at IIT Madras to predict recipe ratings using various data science techniques.

View Notebook

Project Overview

This project focuses on predicting recipe ratings using machine learning techniques. Through this project, I gained practical experience in data preprocessing, feature engineering, and model training, furthering my understanding of applied data science.

The project involved working with a dataset of recipes and their ratings. I implemented various machine learning models, including MultinomialNB, BernoulliNB, and DecisionTreeClassifier, to predict the ratings based on recipe features and user reviews.

Key achievements include achieving a best submission score of 0.77342 using the MultinomialNB model. The project showcases my skills in data analysis, machine learning, and Python programming.
